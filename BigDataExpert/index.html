빅데이터란

기존 DB관리도구로는 저장할 수 없을 정도의 큰 데이터를 저장하고 결과를 분석하는 기술

과거에는 관리할 수 없는 데이터의 양과 일정하지 못한 형태로 인해 쓰지 못하고 버려지는 데이터들이 많았다. 하지만 현재에 들어서 하드웨어와 소프트웨어 양쪽 모두에 발달로 인해 큰 사이즈의 데이터도 몇 시간이면 처리할 수 있게 되었다.

빅데이터

빅데이터 예측 분석의 핵심요소 6가지

1. 조사

관심 대상 집단에서 조사한 결과로부터 일반성을 찾아낸 이후 불확실한 사실에 대한 결론을 예측하는 통계로 활용한다.

2. BRMS

인간의 사고방식을 모방한 BRMS로 지능적 비즈니스 로직 처리와 지능형 시스템을 개발한다.

3. 데이터 시각화

데이터 분석 결과를 쉽게 이해하도록 시각화 자료로 정보를 명확하게 전달해준다.

4. 비정형 데이터

비정형 데이터에서 의미있는 정보를 찾아낸 후 다른 정보와의 연개성을 파악하고, 텍스트의 카테고리를 찾아낸다.

5. 분석

기존의 통계 분석으로는 접근할 수 없는 관계와 상호작용을 분석. 거시적·미시적 관계 패턴을 파악하는 소셜 네트워크 분석을 활용한다.

6. 학습

컴퓨터에 사람이 직접 명시적으로 로직을 지시하지 않아도 데이터를 이용해 컴퓨터가 학습한다.

빅데이터의 원리

(Principle)

규모

데이터가 특정 용량 크기가 되었을 때
데이터가 컴퓨터에서 처리할 수 없을 때
데이터가 Excel에서 열어볼 수 없을 때
데이터의 샘플 수가 많을 때

다양성

데이터는 정형, 반정형, 비정형으로 구분
정형: 고정된 필드에 저장되는 데이터
반정형: 정형의 반대, xml과 html로 구성
비정형: 사진, 동영상의 데이터, 위치기록

속도

오늘날 디지털 데이터는 매우 빠른 속도로 생성되기 때문에 데이터의 수집, 저장, 분석 등이 실시간으로 처리돼야 하지만 모든 데이터가 실시간 처리만을 요구하는 것은 아니다.
